# Journey
Journey is a narrative- & exploration-driven miniRPG: explore a map, fight against enemies and get to know NPCs and the character you've chosen.

## Assignment
Standalone 2D or 3D Game with Clear Core Mechanics <i>(Gameplay & Mechanics Focus)</i>
<b>Project Idea</b>
Development of a standalone game that focuses on clearly defined core mechanics.

focus:
• Clarity
• Gameplay feel
• Clean gameplay loop
<i>Not on scope, but on quality.</i>

<b>Goal</b>

• Development of a playable, self-contained prototype
• Clear objectives and loss conditions
• Comprehensible game rules

<b>Mandatory Content</b>
• Implementation as a 2D or 3D game
• At least one complete gameplay loop:
  o Start → Game → End
• Storage of game or round statistics in a database <i>(not neccesary</i>
• Structured code and clean project organization
